# autocuda plugin

The autocuda plugin for Claude Code. Ships the `autocuda:discover-cuda` and
`autocuda:optimize-cuda` skills: one to map an unfamiliar CUDA project and
produce `project-layout.md`, one to run the autonomous optimization trial loop
against that layout.

For installation, usage, and the example CUDA benchmarks the skills were
developed against, see the main repo README:
<https://github.com/brycelelbach/autocuda>
